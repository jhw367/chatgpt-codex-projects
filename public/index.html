Mobile-first blueprint

LLM Knowledge Base die werkt op iPhone 14 Pro Max

Dit dashboard vertaalt je workflow naar een concrete architectuur: ingest → compile → Q&A → output → health checks, met een stack die je ook mobiel prettig kunt gebruiken.

1) Architectuur (minimaal product)

  1. raw/: bronmateriaal (artikelen, PDF, datasets, repo-notes, afbeeldingen).
  2. compile/: LLM converteert naar wiki-artikelen met backlinks en indexen.
  3. query/: agent voert vragen uit en schrijft antwoorden terug als .md/.marp/.png.
  4. lint/: quality checks op inconsistenties, ontbrekende metadata en nieuwe kansen.
  5. publish/: statische viewer + API, bereikbaar op je iPhone.
