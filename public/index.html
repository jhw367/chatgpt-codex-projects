ChatGPT op afstand

Gebruik deze eenvoudige interface om via je eigen OpenAI-sleutel gesprekken te voeren met ChatGPT. Je volledige gesprek blijft lokaal in je browser opgeslagen.

Gespreksinstellingen

Dit bericht bepaalt de context van het gesprek. Laat het leeg voor de standaardinstructie.

Een lagere waarde maakt de antwoorden gerichter, een hogere waarde creatiever.
